Multiple Files is addon for Contact Form 7

== Description ==

It is add multiple file upload capability to Contact Form 7. You may also attached the files to the sent email. You will reveive all files same. 

== Installation ==

1. Upload the entire `multiple-files-for-contact-form-7` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the 'multiplefile' tag as you create your form.
